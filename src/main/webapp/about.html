About PAL

Below you can find my report describing the system. Here is the abstract:

PAL is a Question Answering (QA) system for Linked Data. It is based on Hakimov's previous work as described in the paper Semantic Question Answering System over Linked Data using Relational Patterns. It was developed during my participation in the Question Answering Systems Project at the Language Technology Lab at TU Darmstadt. As described by Hakimov, processing of natural-language (NL) questions is achieved using Stanford dependencies. Triples are generated from the dependency tree, and then combined to a SPARQL-like pseudo query. NL elements are mapped to ontology elements using WordNet similarity measures, and the solution is selected using different scoring mechanisms. This report also introduces a web frontend that allows for interaction with the system. Evaluation is performed using the QALD-2 challenge.
